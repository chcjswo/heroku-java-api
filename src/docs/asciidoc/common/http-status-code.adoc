[[http-status-code]]
== Http Status Code
[cols="4,6"]
|===
| Status code | Usage

| `200 OK`
| 요청 성공

| `201 Created`
| POST 요청 성공.

| `204 No Content`
| DELETE 요청 성공

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 비인증 상태

| `403 Forbidden`
| 권한 없음

| `404 Not Found`
| 존재 하지 않는 요청 리소스

| `409 Conflict`
| 요청 충돌

| `500 Internal Sever Error`
| 예상치 못한 내부 서버 에러
|===
